# 桌面版开发副本说明

这是用于 Electron / 桌面版封装的开发副本。

用途：
- 后续桌面 App 改造都在这里做
- 原来的 `C:\Users\11370\Desktop\背书软件` 保持作为可用网页版

第一次使用：
```powershell
cd "C:\Users\11370\Desktop\背书软件-桌面版开发"
npm.cmd install
npm.cmd run dev
```

注意：
- 这个副本没有包含 node_modules，需要重新安装依赖。
- 桌面版打包前，先确认网页版副本仍能正常运行。

## 桌面版命令

安装桌面依赖：

```powershell
npm.cmd install
```

开发模式启动桌面窗口：

```powershell
npm.cmd run desktop:dev
```

生成可运行的桌面版目录：

```powershell
npm.cmd run desktop:dir
```

生成便携版 exe：

```powershell
npm.cmd run desktop:dist
```

打包结果会输出到 `release` 文件夹。
